# Nadamark

Nadamark is a bookmark manager for minimalists. You can use the provided docker-compose.yml to self-host it on your own server/computer (coming soon).

## Features

* Fast
* Minimal
* Self-hostable

What more could you want?

## Install with Docker Compose

The easiest way to install Nadamark is with Docker.
